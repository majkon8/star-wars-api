Starship
Object

A Star Wars starship.

Fields

#

name

String !
Non-null

The name of this starship. The common name, such as "Death Star".

model

String !
Non-null

The model or official name of this starship. Such as "T-65 X-wing" or "DS-1 Orbital Battle Station".

manufacturer

String !
Non-null

The manufacturer of this starship. Comma separated if more than one.

cost_in_credits

String !
Non-null

The cost of this starship new, in galactic credits.

length

String !
Non-null

The length of this starship in meters.

max_atmosphering_speed

String !
Non-null

The maximum speed of this starship in the atmosphere. "N/A" if this starship is incapable of atmospheric flight.

crew

String !
Non-null

The number of personnel needed to run or pilot this starship.

passengers

String !
Non-null

The number of non-essential people this starship can transport.

cargo_capacity

String !
Non-null

The maximum number of kilograms that this starship can transport.

consumables

String !
Non-null

The maximum length of time that this starship can provide consumables for its entire crew without having to resupply.

hyperdrive_rating

String !
Non-null

The class of this starships hyperdrive.

MGLT

String !
Non-null

The Maximum number of Megalights this starship can travel in a standard hour. A "Megalight" is a standard unit of distance and has never been defined before within the Star Wars universe. This figure is only really useful for measuring the difference in speed of starships. We can assume it is similar to AU, the distance between our Sun (Sol) and Earth.

starship_class

String !
Non-null

The class of this starship, such as "Starfighter" or "Deep Space Mobile Battlestation"

pilots

[ Person !]!
Non-null

An array of people that this starship has been piloted by.

films

[ Film !]!
Non-null

An array of films that this starship has appeared in.
